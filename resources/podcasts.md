# Podcasts

Software engineering podcasts for learning on the go.

---

| Name | Host(s) | Link | Description |
|------|---------|------|-------------|
| Software Engineering Radio | Various | [se-radio.net](https://www.se-radio.net/) | Long-running podcast covering a wide range of SE topics with expert guests. Great for deepening your understanding. |
| The Changelog | Adam Stacoviak, Jerod Santo | [changelog.com/podcast](https://changelog.com/podcast) | Conversations with open-source developers and maintainers. Good exposure to real-world software projects. |
| CodeNewbie | Saron Yitbarek | [codenewbie.org/podcast](https://www.codenewbie.org/podcast) | Beginner-friendly stories from people learning to code. Motivating for students starting their SE journey. |
| Syntax | Wes Bos, Scott Tolinski | [syntax.fm](https://syntax.fm/) | Web development topics (JavaScript, CSS, Node.js, React). Directly relevant to SE project tech stacks. |
| Software Engineering Daily | Various | [softwareengineeringdaily.com](https://softwareengineeringdaily.com/) | Daily technical interviews covering a broad range of SE topics — architecture, DevOps, AI, and more. |
