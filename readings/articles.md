## Software Design

| Title | Author / Source | Topic | Why Read It |
|-------|----------------|-------|-------------|
| [Is Design Dead?](https://martinfowler.com/articles/designDead.html) | Martin Fowler | Design & Agile | Explores how evolutionary design works in Agile contexts — reconciles upfront design with iterative development. |
| [The Twelve-Factor App](https://12factor.net/) | Adam Wiggins (Heroku) | Architecture | Methodology for building modern, scalable web applications. Directly applicable to SE projects. |
| [Things You Should Never Do, Part I](https://www.joelonsoftware.com/2000/04/06/things-you-should-never-do-part-i/) | Joel Spolsky | Design Decisions | Classic essay on why rewriting code from scratch is almost always a mistake. |
